Lista de Conversas NOVO
Header mais limpo: Titulo "Conversas" com gradiente da marca, contador discreto, botao de busca redondo.
Filtros por status: Chips de Todas / Expirando / Reveladas com dots animados. Facil de filtrar conversas urgentes.
Cards de conversa redesenhados: Layout mais limpo estilo WhatsApp. Avatar maior (52px), nome em destaque, nick abaixo, preview da ultima msg. Sem aquela borda lateral que ficava pesada.
Badge de unread: Bolinha com gradiente laranja/rosa com numero de mensagens nao lidas. Muito mais visivel que o ponto simples anterior.
Online indicator: Bolinha verde no avatar mostrando quem esta online agora.
Timer reorganizado: Pill de tempo alinhada a direita com cores mais claras (verde/amarelo/vermelho piscando).
Tela de Chat NOVO
Header compacto: Botao voltar (chevron), avatar, nome + status online, timer com icone de relogio. Removido o botao circular gigante de voltar (era 52x52, agora e 36x36 com chevron).
Barra de conexao: Frase do encontro + streak resumido numa barra discreta. Nao ocupa espaco visual desnecessario.
Mensagens estilo WhatsApp: Baloes mais arredondados (18px radius), gradiente sutil nas msgs enviadas (laranja->rosa), msgs recebidas em cinza escuro. Mais contraste e legibilidade.
Typing indicator redesenhado: Balao com dots bouncing mais suave. Aparece como se fosse uma mensagem do outro lado.
Quick phrases: Chips arredondados (pill shape) mais tappable, scroll horizontal suave. Estilo stories do Instagram.
Bottom bar totalmente novo: Botao de attach (+ redondo), input com border-radius 22px, botao de enviar com gradiente e animacao de aparecer. Reveal e Pedir ID como pills acima do input - sempre visiveis sem precisar abrir menu.
Menu de acoes (Foto, Presente, etc): Grid 4 colunas com icones maiores (44px), labels, bordas arredondadas. Abre pra cima com animacao suave.
Background sutil: Radial gradients muito sutis (laranja/rosa) pra dar profundidade sem poluir.
O que NAO muda
Toda a logica de JavaScript continua igual: Socket.IO, envio de msgs, ephemeral, typing, pulse, streak, timer, etc. So muda CSS + estrutura HTML.
As funcoes onclick continuam chamando as mesmas funcoes: openChat(), sendMessage(), sendPulse(), getHoroscope(), togglePlusMenu(), etc.
Cores da marca continuam: laranja #ff6b35, rosa #ff3d71, verde #34d399, fundo escuro. Apenas mais vibrante e organizado.
