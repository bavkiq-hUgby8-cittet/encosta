Dicas de Impressao

Papel para Tent Cards

Use papel couce 300g para tent cards e cartoes para um acabamento profissional

Banners em Lona

Para banners, procure uma grafica local com impressao em lona para maior durabilidade

Adesivos Vinil

Adesivos: use vinil adesivo para maior durabilidade em ambientes externos

Resolucao Minima

Sempre use resolucao minima de 300 DPI para garantir qualidade profissional na impressao

Cores Vibrantes

Para cores mais vibrantes, solicite impressao em CMYK em vez de RGB

Margem de Seguranca

Deixe pelo menos 5mm de margem em todos os lados para evitar cortes desalinhados
