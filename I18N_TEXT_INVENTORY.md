## IMPORTANT NOTES FOR TRANSLATORS

1. **Emojis Preserved**: All emojis in toast messages (✅, ⏳, ✉, ⚠️, etc.) should remain in translated versions
2. **Brand Terms**: "Touch?", "Encosta", "Toque" are branding elements - maintain consistency
3. **Numbers/Formats**: 
   - R$ for currency (Brazilian Real)
   - DD/MM format for dates
   - Phone format: (XX) 9XXXX-XXXX
   - CPF format: XXX.XXX.XXX-XX
4. **Tone**: The app uses:
   - **Friendly/Casual** for main UI ("Tudo nasce no gesto")
   - **Technical/Clear** for errors ("Código inválido")
   - **Encouraging** for feedback ("Email já verificado! ✅")
5. **Newline Handling**: Some long messages may contain `\n` - preserve these
6. **HTML Entities**: Some text contains HTML entities (`&nbsp;`, `&#x2190;`) - these are already decoded
