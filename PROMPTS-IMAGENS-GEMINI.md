# Prompts para Geração de Imagens - Touch App - Gemini

## Instruções Técnicas
- Modelo: Gemini Image Generation (native model)
- Gerar no chat Gemini usando o prompt exato abaixo
- Aspect Ratio: 16:9 para Hero, 1:1 para Cards
- Estilo: Fotorrealista, cinematico, qualidade alta
- Aplicacao: Website do Touch App

## POSICAO CORRETA DOS CELULARES (MUITO IMPORTANTE)
Os celulares devem estar com as TELAS VIRADAS UMA PARA A OUTRA (screen-to-screen).
Cada pessoa segura o celular na vertical, com a tela apontando para o outro celular.
As costas dos celulares ficam viradas para fora (para o espectador/camera).
Os alto-falantes ficam na parte de baixo. E como se os celulares estivessem
"se olhando" - tela contra tela, paralelos, proximos um do outro.
NAO e topo-com-topo, NAO e costas-com-costas, NAO e lado a lado com telas para cima.
Todos os prompts abaixo ja refletem essa posicao.
